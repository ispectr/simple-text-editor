# simple-text-editor

### Installation
 
1. Clone the repo
   ```sh
   git clone https://github.com/your_username_/Project-Name.git
   ```
2. Install.
   ```sh
   mkdir build && cd build
   cmake .. -DCMAKE_INSTALL_PREFIX=$HOME/.local-kde
   make install
   source prefix.sh # located in the build directory
   ```
